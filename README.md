# CODSOFT Internship - Task 2

## Rule-Based Chatbot

### Overview

This project was completed as part of the CODSOFT Artificial Intelligence Internship.

The project is a Rule-Based Customer Support Chatbot developed using Python and Tkinter GUI. The chatbot is designed for Scooby Pet Hostel and responds to customer queries using predefined rules and keyword matching.

### Features

* Graphical User Interface (GUI) using Tkinter
* Rule-Based Chatbot System
* Customer Support Automation
* Boarding Charges Information
* Service Information
* Location Details
* Booking Assistance
* Pet Policy Information
* Dogs-Only Boarding Support
* Aggressive Pet Handling Policy
* AC Accommodation Information
* No Cages and No Tying Policy
* Human-Friendly Responses

### Technologies Used

* Python
* Tkinter

### How to Run

1. Clone the repository
2. Open terminal in project folder
3. Run:

python scooby_pet_hostel_gui.py
